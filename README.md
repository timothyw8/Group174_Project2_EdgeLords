How To Use this Program

1. Start in main.cpp, click Run to start the program, a gray pop-up window should appear

2. Click on the textbox next to "From: ", and type in a city with proper capitalization (e.g. "Miami" not "miami"), this is the city the algorithms start at

3. Repeat the process in step 3 for the textbox next to "To: "

5. Click the dropdown menu next to "Algorithm", and select an algorithm

6. Click the green "Calculate" button to start the algorithm

7. "Minimum delivery cost: " should appear on the screen, which represents the least costly route to get from the city inputed next to "From: " to "To: " in dollars

8. "Execution time: "  should appear right below, which represents the time it took for the program to execute the selected algorithm

9. Close the window to stop the program
